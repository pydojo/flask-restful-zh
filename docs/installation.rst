.. _installation:

安装
============

.. currentmodule:: flask_restful

安装 Flask-RESTful 用 ``pip`` ::

    pip install flask-restful


开发版本可以下载自 ` GitHub 仓库 <https://github.com/flask-restful/flask-restful>`_ ::

    git clone https://github.com/flask-restful/flask-restful.git
    cd flask-restful
    python setup.py develop


Flask-RESTful 扩展件的依赖包 (会自动随 ``pip`` 自动安装):


* `Flask <http://flask.pocoo.org>`_ 版本要在 0.10 以上

Flask-RESTful 需要运行在 Python 版本 2.7, 3.4, 3.5, 3.6, 3.7
